# ⚠️ คำเตือนความเสี่ยง & ข้อสงวนสิทธิ์

> **กรุณาอ่านให้ครบก่อนศึกษาเอกสารนี้**

---

## ลักษณะของเอกสารนี้

เอกสารนี้ (ต่อไปนี้เรียก "White Paper") เป็น **เอกสารให้ข้อมูล** ที่อธิบายการออกแบบทางเทคนิคและแนวคิดของ Matsuri Coin (MTC) และ Matsuri Protocol

**เอกสารนี้ไม่ใช่สิ่งต่อไปนี้:**
- เอกสารเสนอขายหรือจำหน่ายหลักทรัพย์
- การชักชวนลงทุนหรือให้คำปรึกษาด้านการลงทุน
- การรับประกันกำไรหรือรายได้
- สัญญาที่มีผลผูกพันทางกฎหมาย

---

## ความเสี่ยงเกี่ยวกับ MTC

การได้มา ถือครอง ใช้งาน MTC มีความเสี่ยงรวมถึงแต่ไม่จำกัดเพียงต่อไปนี้

### ความเสี่ยงด้านราคาผันผวน
ราคา MTC ผันผวนตามอุปสงค์-อุปทานในตลาด **อาจต่ำกว่าราคาที่ซื้อมาอย่างมาก** กรณีเลวร้ายที่สุดมูลค่าอาจเป็นศูนย์ กลไกทางเศรษฐกิจที่ระบุใน White Paper นี้ (Flywheel, Buy-back ฯลฯ) อธิบายโครงสร้างการออกแบบ **ไม่ได้รับประกันว่าราคาจะขึ้น**

### ความเสี่ยงด้านสภาพคล่อง
ปัจจุบัน MTC ซื้อขายได้บน Raydium (DEX) แต่หากสภาพคล่องไม่เพียงพอ **อาจไม่สามารถขายที่ราคา/เวลาที่ต้องการได้**

### ความเสี่ยงทางเทคนิค
- Smart Contract อาจมีบั๊กหรือช่องโหว่
- อาจไม่สามารถทำธุรกรรมได้ชั่วคราวเนื่องจาก Solana Network ขัดข้อง/หยุดทำงาน
- อาจเกิดความไม่สอดคล้องของข้อมูลช่วงย้าย off-chain → on-chain

### ความเสี่ยงด้านกฎระเบียบ
กฎหมายเกี่ยวกับคริปโตเปลี่ยนแปลงเร็วในแต่ละประเทศ การเปลี่ยนแปลงกฎในอนาคตอาจทำให้การซื้อขาย ใช้งาน ถือครอง MTC **ถูกจำกัดหรือห้าม**

### ความเสี่ยงด้านโครงการ
- ฟังก์ชันและกำหนดการที่ระบุใน White Paper อาจเปลี่ยนแปลง
- โปรเจกต์อาจไม่ดำเนินไปตามแผน หรืออาจยุติ
- การเปลี่ยนแปลงโครงสร้างทีมพัฒนาอาจกระทบ Roadmap

### ความเสี่ยงด้าน Private Key / Wallet
หากสูญหาย Private Key หรือ Seed Phrase ของ Wallet **ไม่มีวิธีกู้คืน MTC** เราไม่เก็บ Private Key ของผู้ใช้ และไม่สามารถช่วยกู้คืนได้

---

## สิ่งที่เราไม่ทำ

| รายการ | คำอธิบาย |
| :--- | :--- |
| **แลกเปลี่ยน/แปลงคริปโต** | เราไม่ได้จดทะเบียนเป็นผู้ประกอบธุรกิจแลกเปลี่ยนคริปโต ไม่ให้บริการแลก MTC กับเงินตราโดยตรงเลย |
| **คำปรึกษาการลงทุน** | เนื้อหาในเอกสารนี้ไม่ใช่คำแนะนำการลงทุน การตัดสินใจซื้อ MTC โปรดทำด้วยความรับผิดชอบของตัวเอง |
| **รับประกันเงินต้น** | ไม่รับประกันการรักษา/เพิ่มมูลค่าของ MTC |
| **บริหาร Private Key** | ไม่เก็บหรือบริหาร Private Key ของ Wallet ผู้ใช้ |

## ข้อควรระวังเกี่ยวกับการคาดการณ์อนาคต

เอกสารนี้มีข้อความเกี่ยวกับ "การคาดการณ์อนาคต" ถ้อยคำเช่น "มุ่งหวัง" "เตรียม" "แนวคิด" "ออกแบบ" เป็นต้น แสดงถึงแผนการและเป้าหมายในอนาคต ไม่ใช่ข้อเท็จจริงที่แน่นอน ผลลัพธ์จริงอาจแตกต่างจากที่ระบุอย่างมาก ตามสภาพตลาด ปัญหาทางเทคนิค สภาพกฎหมาย และปัจจัยอื่นๆ

---

## ภูมิภาคที่เกี่ยวข้อง

เอกสารนี้ไม่ได้มุ่งเป้าไปที่ผู้อยู่อาศัยในภูมิภาคที่การได้มา/ใช้งาน MTC ถูกกฎหมายห้าม กรุณาตรวจสอบกฎหมายของประเทศ/ภูมิภาคที่คุณอาศัยอยู่ และตัดสินใจด้วยความรับผิดชอบของตัวเอง

---
