# ⛩️ Matsuri Coin (MTC) मा स्वागत छ

> **和 (wa — सद्भाव) को पुल: प्रतिस्पर्धाबाट सह-सिर्जनासम्म।**

तपाईंले जापानी संस्कृतिको अनुभव गर्नुहोस्, यसलाई पुर्‍याउनुहोस्, वा यसको रक्षा गर्नुहोस् —
हरेक प्रकारको संलग्नता आर्थिक मूल्य बन्छ, संस्कृतिको भविष्यलाई थाम्न।
यही **Matsuri Coin (MTC)** हो।
