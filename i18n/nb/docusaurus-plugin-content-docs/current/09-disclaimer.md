# ⚠️ Risikoadvarsel og ansvarsfraskrivelse

> **Les dette grundig før du leser resten av dokumentet.**

---

## Dokumentets karakter

Dette dokumentet («whitepaperet») er et **informasjonsmateriale** som beskriver det tekniske designet og visjonen for Matsuri Coin (MTC) og Matsuri Protocol.

**Dokumentet utgjør ikke noe av følgende:**
- Et dokument i forbindelse med offentlig tilbud eller salg av verdipapirer
- Investeringsrådgivning eller oppfordring til å investere
- En garanti for avkastning eller inntjening
- En juridisk bindende avtale

---

## Risiko ved MTC

Å erverve, eie og bruke MTC innebærer (herunder, men ikke begrenset til) følgende risikoer.

### Prisrisiko
MTCs pris svinger med markedets tilbud og etterspørsel og kan **falle betydelig under anskaffelsesprisen**. I verste fall kan verdien gå til null. De økonomiske mekanismene beskrevet i whitepaperet (svinghjulet, tilbakekjøp osv.) beskriver designstrukturen og **garanterer ikke prisstigninger**.

### Likviditetsrisiko
MTC kan i dag handles på Raydium (DEX), men hvis likviditeten er utilstrekkelig, **kan det være umulig å selge til ønsket pris og tidspunkt**.

### Teknisk risiko
- Smart contracts kan inneholde feil eller sårbarheter
- Feil eller nedetid i selve Solana-nettverket kan midlertidig gjøre transaksjoner umulig
- Dataavvik kan oppstå i forbindelse med off-chain → on-chain-migrering

### Reguleringsrisiko
Lovgivningen om kryptoaktiva endrer seg raskt i alle land. Framtidige regelendringer kan **begrense eller forby** handel, bruk eller besittelse av MTC.

### Prosjektrisiko
- Funksjoner og tidsplaner i whitepaperet kan endres
- Prosjektet kan gå saktere enn planlagt eller bli innstilt
- Endringer i utviklingsteamet kan påvirke veikartet

### Privatnøkkel / lommebok-risiko
Hvis du mister lommebokens privatnøkkel eller seed-fraser, **finnes det ingen måte å gjenopprette MTC-en din på**. Vi oppbevarer ikke brukerens privatnøkkel og kan ikke hjelpe med gjenoppretting.

## Hva vi IKKE gjør

| Punkt | Forklaring |
| :--- | :--- |
| **Veksling av krypto** | Vi er ikke registrert som krypto-vekslingsvirksomhet. Vi tilbyr ikke direkte veksling mellom MTC og fiat |
| **Investeringsrådgivning** | Innholdet i dokumentet er ikke investeringsrådgivning. Beslutning om kjøp av MTC er ditt eget ansvar |
| **Kapitalgaranti** | Vi garanterer ikke at MTCs verdi opprettholdes eller stiger |
| **Nøkkelforvaltning** | Vi oppbevarer eller forvalter ikke brukerens lommebok-privatnøkler |

## Om framtidsrettede utsagn

Dokumentet inneholder framtidsrettede utsagn. Uttrykk som «sikter mot», «planlagt», «visjon» og «design» angir framtidige planer og mål, ikke endelige fakta. Faktiske resultater kan avvike vesentlig pga. markedsforhold, tekniske utfordringer, regulering og andre forhold.

---

## Målgruppe-regioner

Dokumentet retter seg ikke mot personer bosatt i regioner der erverv og bruk av MTC er forbudt ved lov. Sjekk lovgivningen i ditt eget land/region og ta beslutningen på eget ansvar.
