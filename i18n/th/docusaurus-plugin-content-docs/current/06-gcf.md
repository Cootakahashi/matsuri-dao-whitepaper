# 🤝 GCF — ชุมชนนานาชาติที่เป็นแกนกลางของการร่วมสร้าง

> จาก "ผู้ชม" สู่ "ผู้ร่วมสร้าง"

วิสัยทัศน์ของ Matsuri Coin ที่ว่า "จากการแข่งขันสู่การร่วมสร้าง" ไม่สามารถบรรลุได้ด้วยเทคโนโลยีและโทเค็นเพียงอย่างเดียว
ผู้คนที่เต็มไปด้วยแรงบันดาลใจซึ่งกระจายตัวอยู่ทั่วโลก เชื่อมโยงวัฒนธรรมและผู้คนในพื้นที่ของตน ทำให้ระบบนิเวศมีชีวิต — นั่นคือ **GCF (Global Community Friends)**

GCF คือ **ชุมชนเฉพาะผู้ได้รับเชิญ** ที่มีส่วนร่วมในการดำเนินงานระบบนิเวศของ Matsuri
สมาชิกจัดประสบการณ์วัฒนธรรมในพื้นที่ของตนเอง และช่วยให้ระบบนิเวศเติบโตจากภาคสนาม
