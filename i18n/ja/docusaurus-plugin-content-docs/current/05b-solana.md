# ⚡ なぜSolanaなのか？

> **Ethereumへの敬意とともに——MTCのユースケースにはSolanaが必要だった。**

Ethereumは、スマートコントラクトという革命を世界にもたらしたブロックチェーンの先駆者です。DeFiやNFTの文化を切り拓き、最大級の開発者コミュニティと長年の実績が築いた**セキュリティと信頼性**は、今なお業界随一です。

しかし、MTCが実現したいのは「観光の現場で使える日常の決済」。お賽銭のように100円を送る行為に、数百円の手数料や数十秒の待ち時間は許容できません。

---

## 比較：Ethereum vs Solana

| 比較項目 | Ethereum | Solana |
| :--- | :---: | :---: |
| **送金手数料** | 数百〜数千円 | **約0.04円** |
| **処理速度** | 12秒〜数分 | **0.4秒** |
| **秒間処理能力** | 約15件 | **数千件** |
| **得意領域** | 高額取引・DeFi・セキュリティ重視の用途 | **高頻度・少額決済・リアルタイム体験** |

![亀と新幹線 ── Ethereumの確かな歩みと、Solanaの圧倒的な速度](/brand/08_A_tortoise_shinkansen.webp)

Ethereumが築いた土台がなければ、Solanaも今のWeb3も存在しません。
その上で、**観光という実需 × 高頻度のマイクロ決済**という要件に応えられるのは、現時点ではSolanaだけでした。
