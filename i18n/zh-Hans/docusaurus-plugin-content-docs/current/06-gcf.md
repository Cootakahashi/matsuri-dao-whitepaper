# 🤝 GCF——共创核心的国际社区

> 从"观众"到"共创者"。

Matsuri Coin 所倡导的"从竞争到共创"愿景，仅靠技术与代币无法实现。
散布在世界各地的**充满热情的人们**，在各自的所在地连接文化与人，让生态系统脉动起来——这就是 **GCF（Global Community Friends）**。

GCF 是参与运营 Matsuri 生态系统的**邀请制社区**。
成员在各自的地区策划与运营文化体验，从现场培育生态系统。
