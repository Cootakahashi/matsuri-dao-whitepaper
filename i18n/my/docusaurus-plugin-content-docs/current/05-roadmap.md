# 🗓️ Roadmap နှင့် အုပ်ချုပ်ရေး

> **ယုံကြည်စိတ်ချရမှုသို့ လမ်းကြောင်း**
> ဤသည်မှာ ရေတိုစပ်ယူလေးရှင်း ပရောဂျက်မဟုတ်ပါ။
> **အဓိက ပလက်ဖောင်း ဖွံ့ဖြိုးမှု ပြီးစီးပြီးဖြစ်သည်** — ကျွန်ုပ်တို့သည် scaling အဆင့်တွင် ရှိသည်။

## မဟာဗျူဟာ မိုင်တိုင်များ

### 🔥 Phase 1: နိုးထခြင်း (2026 H1 — ယခု)

**ခေါင်းစဉ်: အခြေခံတည်ဆောက်မှုနှင့် ငွေကြေးစီးဆင်းမှု**

ထုတ်ကုန် တည်ဆောက်ပြီးပြီ။ CEO ဦးဆောင်သော ငွေကြေးစနစ်ဖြင့် ဝင်ငွေရှာခြင်းနှင့် ကနဦး liquidity ရရှိရေးတွင် အာရုံစိုက်သည်။

| အခြေအနေ | မိုင်တိုင် | အသေးစိတ် |
| :---: | :--- | :--- |
| ✅ | **ထုတ်ကုန် ထွက်ရှိ** | Matsuri Webapp နှင့် GCF Admin Dashboard လည်ပတ်နေပြီ |
| ✅ | **ငွေပေးချေမှုနှင့် ကြီးထွားမှု** | MTC ပေးချေမှု + referral airdrop features ဆောင်ရွက်ပြီး |
| ✅ | **မီဒီယာ ထွက်ရှိ** | J-Times (Web & Podcast) ဖြန့်ဝေမှု ပြင်ဆင်ပြီး |
| ✅ | **Genesis** | Solana ပေါ်တွင် MTC Token Generation Event |
| ✅ | **Liquidity** | Raydium တွင် ကနဦး LP pool ဖန်တီးပြီး |
| ⬜ | **Incentive Program** | 50% target-APY liquidity mining စတင်ရန် |
| ⬜ | **System Go-Live** | Solana MEV/arbitrage bot production တွင် |
| ⬜ | **VIP စုဆောင်းမှု** | ပထမ GCF VIP အဖွဲ့ဝင် ၂၀ ဦး ရွေးချယ်ပြီး |

**ခေါင်းစဉ်: စစ်မှန်သော ပိုင်ဆိုင်မှုများနှင့် Adventure Mining**

| အခြေအနေ | မိုင်တိုင် | အသေးစိတ် |
| :---: | :--- | :--- |

| ⬜ | **ကမ္ဘာလုံး ချဲ့ထွင်** | အာရှတစ်ဝှမ်း (ထိုင်း, ထိုင်ဝမ် စသည်) မိတ်ဖက် |
| ⬜ | **ပိုင်ဆိုင်မှု စီမံခန့်ခွဲ** | အိမ်ခြံမြေ, ရှယ်ယာ, crypto ပိုင်ဆိုင်မှု တည်ဆောက် |

### 🌊 Phase 3: ပတ်လည်လည်ခြင်း (2027+)

**ခေါင်းစဉ်: အကြီးစား ချဲ့ထွင်မှု၊ ပူးတွဲဖန်တီးမှု စီးပွားရေးနှင့် ဗဟိုချုပ်ကိုင်မှု လျှော့ခြင်း**

အများပြည်သူ ထွက်ရှိမှု၊ on-chain ဈေးကွက်နှင့် ဂေဟစနစ် အပြည့်အဝ လည်ပတ်ခြင်း။

## 🏛️ အုပ်ချုပ်ရေး (DAO)

Matsuri Protocol သည် တဖြည်းဖြည်း **ဗဟိုချုပ်ကိုင်မှု ကင်းသော အလိုအလျောက်အဖွဲ့အစည်း (DAO)** သို့ ပြောင်းရွှေ့မည်။
GCF အဖွဲ့ဝင်များ (Platinum/Gold) သည် အဓိကဆုံးဖြတ်ချက်များတွင် **မဲပေးပိုင်ခွင့်** ရှိမည်:

| မဲပေးမှု | နယ်ပယ် |
| :--- | :--- |
| **💰 ရန်ပုံငွေ ခွဲဝေမှု** | မည်သည့် စီးပွားရေးသစ် သို့ marketing ကို ရန်ပုံငွေထောက်ပံ့မည် |
| **⚙️ Protocol အဆင့်မြှင့်** | ကြေးနှုန်းနှင့် mining ဆု curve ကို ချိန်ညှိခြင်း |
| **⛩️ ယဉ်ကျေးမှု အသိအမှတ်ပြု** | မည်သည့် ပွဲတော်များနှင့် ဘုရားကျောင်းများကို "တရားဝင် ဘုရားဖူးနေရာ" အဖြစ် အသိအမှတ်ပြုမည် |
