## MTCに関するリスク

MTCの取得・保有・利用には、以下を含む（ただしこれに限らない）リスクがあります。

### 価格変動リスク
MTCの価格は市場の需給により変動し、**取得時の価格を大幅に下回る可能性**があります。最悪の場合、価値がゼロになる可能性もあります。本ホワイトペーパーに記載された経済メカニズム（フライホイール、買い戻し等）は設計上の構造を説明するものであり、**価格の上昇を保証するものではありません。**

### 流動性リスク
MTCは現在Raydium（DEX）で取引可能ですが、流動性が十分でない場合、**希望する価格・タイミングでの売却ができない可能性**があります。

### 技術リスク
- スマートコントラクトにバグ・脆弱性が含まれる可能性があります
- Solanaネットワーク自体の障害・停止により、取引が一時的に不可能になる可能性があります
- オフチェーン→オンチェーン移行時にデータの不整合が生じる可能性があります

### 規制リスク
暗号資産に関する法規制は各国で急速に変化しています。将来の規制変更により、MTCの取引・利用・保有が**制限または禁止される可能性**があります。

### プロジェクトリスク
- 本ホワイトペーパーに記載された機能・スケジュールは変更される可能性があります
- プロジェクトが計画通りに進行しない、または中止される可能性があります
- 開発チームの体制変更により、ロードマップが影響を受ける可能性があります

### 秘密鍵・ウォレットリスク
ウォレットの秘密鍵やシードフレーズを紛失した場合、**MTCを復元する手段はありません。** 弊社はユーザーの秘密鍵を保管しておらず、復元支援を行うことはできません。

## 弊社が行わないこと

| 項目 | 説明 |
| :--- | :--- |
| **暗号資産の交換・換金** | 弊社は暗号資産交換業の登録を行っていません。MTCと法定通貨の直接交換サービスは一切提供しません |
| **投資助言** | 本ドキュメントの内容は投資助言ではありません。MTC取得の判断はご自身の責任で行ってください |
| **元本保証** | MTCの価値の維持・上昇を保証するものではありません |
| **秘密鍵の管理** | ユーザーのウォレット秘密鍵を保管・管理しません |
