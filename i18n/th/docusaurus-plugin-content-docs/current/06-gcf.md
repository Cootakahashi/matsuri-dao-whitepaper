# 🤝 GCF — ชุมชนระหว่างประเทศที่เป็นแกนกลางของการร่วมสร้าง

> **จาก "ผู้ชม" สู่ "ผู้ร่วมสร้าง"**
> การเข้าร่วม GCF คือการเป็นเพื่อนร่วมสร้างเศรษฐกิจ Matsuri ไปด้วยกัน

วิสัยทัศน์ "จากการแข่งขันสู่การร่วมสร้าง" ของ Matsuri Coin ไม่อาจทำให้เป็นจริงด้วยเทคโนโลยีและโทเคนเพียงอย่างเดียว
**คนที่มีใจรัก**ซึ่งกระจายอยู่ทั่วโลก ในที่ของแต่ละคน เชื่อมวัฒนธรรมและผู้คน ทำให้เลือดไหลเวียนในระบบนิเวศ — **GCF (Global Community Friends)** คือผู้รับผิดชอบแกนกลางนี้

GCF ไม่ใช่แค่ "สมาชิก"
แต่เป็นองค์กรพาร์ทเนอร์ (DAO) **แบบเฉพาะคำเชิญเต็มรูปแบบ** ที่แบ่งปันรายได้ในฐานะ **ชุมชนธุรกิจ** และเข้าร่วมการเติบโตของระบบนิเวศ Matsuri โดยตรง

## สิทธิพิเศษ ①: Mining เชิงปฏิบัติการ (สิทธิ์การขุด)

MTC **550 ล้าน (ประมาณ 61% ของอุปทานรวม)** ที่ปลดล็อกวันที่ 1 มิถุนายน 2027 ไม่ได้ขายในตลาด แต่เก็บไว้เป็น **พูลรางวัลสำหรับผู้ร่วมสร้าง**

:::tip รางวัลตามผลงานล้วนๆ
ตามปริมาณกิจกรรมของคุณ (ยอดขาย, จำนวนคนที่รวบรวม, จำนวนครั้งที่เป็นไกด์) MTC จะถูกจ่ายจากพูลโดยอัตโนมัติ
ทำเท่าไหร่ ได้ผลตอบแทนยุติธรรมเท่านั้น — นั่นคือการออกแบบรางวัลของ GCF
:::

### รางวัลก่อนปลดล็อก Mining Pool (2026〜พ.ค. 2027)

การปล่อย MTC จาก Mining Pool เริ่มมิถุนายน 2027 แต่ **ช่วงก่อนหน้านั้นก็มีรางวัลเกิดขึ้น**:

| ประเภทรางวัล | เนื้อหา | วิธีจ่าย |
| :--- | :--- | :--- |
| **ค่าคอมแนะนำ** | รางวัลแนะนำสมาชิก ทัวร์ อีเวนต์ ฯลฯ | USDC + MTC (ทันที) |
| **ค่าตอบแทนไกด์** | ค่าคอมต่ออีเวนต์ + ทิป | โอนธนาคาร / คริปโต |
| **สะสม Contribution Score** | ทุกกิจกรรมถูกบันทึก off-chain และเมื่อพูลปลดล็อกมิถุนายน 2027 **MTC จะถูกแจกรวบยอดตาม Score ที่สะสมไว้** | แปลงมิถุนายน 2027 |

:::info นั่นคือ ยิ่งเริ่มเร็ว ยิ่งได้เปรียบ
ช่วงก่อนมิถุนายน 2027 คือช่วงสะสม Score สมาชิกที่กิจกรรมมากที่สุดในช่วงนี้จะรับ MTC สูงสุดตอนปลดล็อกพูล ค่าคอมแนะนำและค่าตอบแทนไกด์จ่ายแบบเรียลไทม์ จึง **ไม่มีช่วงที่รายได้เป็นศูนย์**
:::

รายละเอียดกำหนดการ Halving ดู [Tokenomics](/docs/tokenomics)

:::tip ออกแบบให้ผู้ร่วมสร้างช่วงแรกได้ผลตอบแทนมากกว่า
เช่นเดียวกับ Bitcoin ด้วยกลไก Halving **ปริมาณปล่อยในช่วงแรกสูงที่สุด** กลไกที่ผู้เคลื่อนไหวในช่วงที่ 1 (2027〜2029) ได้ MTC มากที่สุด รางวัลไม่ได้ตัดสินด้วย "ความเร็วในการเข้าร่วม" แต่ด้วย **"ปริมาณและคุณภาพของการมีส่วนร่วม"**
:::

## สิทธิพิเศษ ②: ค่าคอมแนะนำสูง

โดยการแนะนำสินค้าราคาสูง (Membership, VIP Tour, อสังหาฯ พันธมิตร ฯลฯ) คุณได้ **ค่าคอมสูง (USDC + MTC)** ที่ต่างชั้นจาก Affiliate ทั่วไป
คืนสู่คุณ **ทันที** ผ่าน Smart Contract — ไม่ผ่านมือคน จึงไม่มีความล่าช้าและความไม่โปร่งใส

การแนะนำไม่ใช่รางวัลครั้งเดียว ตราบใดที่ผู้ที่คุณแนะนำยังเคลื่อนไหวในระบบนิเวศ **การแบ่งรายได้จะเกิดขึ้นอย่างต่อเนื่อง**
นั่นคือ เครือข่ายที่คุณขยาย จะกลายเป็นสินทรัพย์ของคุณเอง

## บทบาทของ GCF — สมาชิกทำอะไร

สมาชิก GCF ไม่ใช่นักลงทุนที่ "ถือไว้เฉยๆ"
แต่เป็นผู้ร่วมสร้างที่เคลื่อนไหวใน **ภาคสนาม** ของระบบนิเวศและสร้างคุณค่า

| กิจกรรม | เนื้อหา | ผลิตภัณฑ์ที่เชื่อมโยง |
| :--- | :--- | :--- |
| **🗺️ เชิญไกด์** | ค้นหาไกด์วัฒนธรรมตามที่ต่างๆ และเชิญเข้าสู่แพลตฟอร์ม Matsuri | [Matsuri App](/docs/ecosystem) |
| **🎪 จัดอีเวนต์** | ออกแบบและลงประกาศอีเวนต์/ทัวร์ประสบการณ์วัฒนธรรมของตัวเอง และรวบรวมคน | [Matsuri App](/docs/ecosystem) |
| **📢 ส่งคอนเทนต์** | ผลิตและกระจายบทความ/พอดแคสต์ J-Times เพื่อดึงดูดผู้ใช้ใหม่ | [J-Times](/docs/ecosystem) |
| **🤝 บริหารชุมชน** | เลี้ยงดูชุมชนท้องถิ่นและออนไลน์ เสริมความสัมพันธ์ระหว่างสมาชิก | GCF Admin Web |
| **📊 Operation** | ติดตามการแนะนำ วิเคราะห์ยอดขาย บริหารทีม ด้วย GCF Admin Dashboard | GCF Admin Web |

ทุกกิจกรรมถูกบันทึก on-chain ในรูปของ **Proof of Action** และเชื่อมโยงโดยตรงกับรางวัล Mining
"ทำสิ่งที่ชอบ" กลายเป็นกิจกรรมทางเศรษฐกิจโดยตรง — นี่คือโมเดลร่วมสร้างของ Matsuri
