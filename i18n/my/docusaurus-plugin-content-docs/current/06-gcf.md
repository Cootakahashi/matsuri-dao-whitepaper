# 🤝 GCF — အတူတကွ ဖန်တီးခြင်း၏ အလယ်ဗဟိုဖြစ်သော နိုင်ငံတကာ အသိုက်အဝန်း

> "ပရိသတ်" မှ "အတူဖန်တီးသူ" သို့။

Matsuri Coin ၏ "ပြိုင်ဆိုင်ခြင်းမှ အတူဖန်တီးခြင်းသို့" ဟူသော အမြင်သည် နည်းပညာနှင့် တိုကင်များဖြင့်သာ အကောင်အထည်ဖော်၍ မရနိုင်ပါ။
ကမ္ဘာတစ်ဝန်း ပျံ့နှံ့နေသော **စိတ်အားထက်သန်သူများ** က မိမိတို့ နေရာများ၌ ယဉ်ကျေးမှုနှင့် လူများကို ဆက်စပ်ပေးကာ ဂေဟစနစ်ကို အသက်ပေးကြသည် — ထိုသည်ပင် **GCF (Global Community Friends)** ဖြစ်သည်။

GCF သည် Matsuri ဂေဟစနစ်အား **ဖိတ်ကြားမှုဖြင့်သာ ဝင်ရောက်နိုင်သော အသိုက်အဝန်း** ဖြစ်ပြီး လုပ်ငန်းဆောင်ရွက်ရာ၌ ပါဝင်လျက်ရှိသည်။
အဖွဲ့ဝင်များသည် မိမိတို့ နယ်မြေတွင် ယဉ်ကျေးမှု အတွေ့အကြုံများကို စီစဉ်ပြီး မြေပြင်ဆုံးမှ ဂေဟစနစ်ကို ကြီးထွားအောင် ထောက်ပံ့ပေးကြသည်။

GCF သည် လုံးဝ ဖိတ်ကြားမှု အခြေပြု အသိုက်အဝန်း ဖြစ်သည်။ အသေးစိတ်ကို လက်ရှိအဖွဲ့ဝင်များမှ မိတ်ဆက်ပေးခံရသူများသို့သာ တစ်ဦးချင်း မျှဝေပါသည်။
:::
