# 🤝 GCF——공창의 중핵을 담당하는 국제 커뮤니티

> **"관객"에서 "공창자"로.**
> GCF에 참여한다는 것은, Matsuri 경제권을 함께 키워 가는 동료가 되는 것입니다.

Matsuri Coin이 내세우는 "경쟁에서 공창으로"라는 비전은, 기술이나 토큰만으로는 실현할 수 없습니다.
세계 곳곳에 흩어진 **열정을 가진 사람들**이, 각자의 장소에서 문화와 사람을 이어 주고, 에코시스템에 피를 통하게 하는——그 중핵을 담당하는 것이 **GCF(글로벌 커뮤니티 프렌즈)** 입니다.

GCF는 단순한 "회원권"이 아닙니다.
**사업의 공동체**로서 수익을 나누고, Matsuri 에코시스템의 성장 그 자체에 참여하는 **완전 초대제** 파트너 조직(DAO)입니다.

## 회원 티어

| 티어 | 역할 | 특권 |
| :---: | :--- | :--- |
| **👑 플래티넘** | 창업 파트너 / VIP | **50명 한정**. 사업 의사 결정에의 참가권, 최고 랭크의 보상 설계 |
| **🥇 골드** | 앰배서더 | 실행 부대. 활동량에 따른 보상. 마이닝과 추천 보상의 환원율이 최대화 |

플래티넘은 문자 그대로 **50명밖에 존재하지 않는** 창업 파트너 분량입니다.
골드 멤버는, 현장에서 움직이고, 이어 주고, 넓혀 가는——에코시스템의 엔진 그 자체입니다.

### 플래티넘 멤버십 가격

플래티넘 분량은 단계적으로 가격이 상승합니다. 조기에 참가할수록, 낮은 가격으로 창업 파트너가 될 수 있습니다.

| 분량 번호 | 가격(세금 별도) |
| :---: | :---: |
| 1〜10 | **50만 엔**(약 460만 원) |
| 11〜20 | **80만 엔**(약 740만 원) |
| 21〜30 | **150만 엔**(약 1,380만 원) |
| 31〜40 | **250만 엔**(약 2,300만 원) |
| 41〜50 | **500만 엔**(약 4,600만 원) |

<small>*※ 각 가격대의 분량이 채워지는 대로, 다음 가격대로 이행합니다.*</small>

## 특전 ①: 실제 활동 마이닝(채굴권)

2027년 6월 1일에 락업이 해제되는 **5억 5,000만 개(총 공급의 약 61%)** 의 MTC는, 시장에 매각되는 것이 아닌 **기여자에의 보상 풀**로서 확보됩니다.

:::tip 완전 성과 보상
당신의 활동량(매출, 집객 수, 가이드 횟수)에 따라, 풀에서 자동으로 MTC가 지급됩니다.
일한 만큼, 정당하게 보답받는다——그것이 GCF의 보상 설계입니다.
:::

### 마이닝 풀 해금 전(2026년〜2027년 5월)의 보상

마이닝 풀의 MTC 방출은 2027년 6월부터지만, **그때까지의 기간에도 보상은 발생합니다**:

| 보상의 종류 | 내용 | 지급 방법 |
| :--- | :--- | :--- |
| **추천 커미션** | 회원권·투어·이벤트 등의 추천 보상 | USDC + MTC(즉시) |
| **가이드 보상** | 이벤트 개최마다의 커미션+팁 | 은행 이체 / 암호화폐 |
| **기여 스코어의 축적** | 모든 활동이 오프체인에서 기록되어, 2027년 6월의 풀 해금 시에 **축적한 스코어에 따라 MTC가 일괄 배포**됩니다 | 2027년 6월에 변환 |

:::info 즉, 빨리 시작할수록 유리
2027년 6월까지는 스코어를 모으는 기간입니다. 이 기간에 가장 활동한 멤버가, 풀 해금 시에 최대의 MTC를 받습니다. 추천 커미션이나 가이드 보상은 실시간으로 지급되기 때문에, **수입 제로의 기간은 없습니다**.
:::

반감기의 자세한 스케줄은 [토크노믹스](/docs/tokenomics)를 보세요.

:::tip 조기 기여자가 더 많이 보답받는 설계
비트코인과 같이, 반감기에 의해 **이른 단계의 방출량이 가장 많아집니다**. 제1기(2027〜2029년)에 적극적으로 활동한 사람이, 가장 많은 MTC를 받을 수 있는 구조입니다. 보상은 "참가의 빠름"이 아닌 **"기여의 양과 질"** 로 결정됩니다.
:::

## 특전 ②: 고액 추천 보상

고단가 상품(회원권, VIP 투어, 제휴 부동산 등)을 추천함으로써, 일반 어필리에이트와는 차원이 다른 **고액 보상(USDC + MTC)** 을 획득할 수 있습니다.
스마트 컨트랙트 경유로 **즉시 환원**——사람의 손을 거치지 않기 때문에, 지연도 불투명함도 없습니다.

추천은 한 번만의 보상이 아닌, 추천한 상대가 에코시스템 내에서 활동을 계속하는 한, **지속적인 수익 분배**가 발생합니다.
즉, 당신이 넓힌 네트워크가, 당신 자신의 자산이 되는 것입니다.

## GCF의 역할——멤버는 무엇을 하는가

GCF 멤버는 "보유하고 기다리기만 하는" 투자자가 아닙니다.
에코시스템의 **현장**에서 움직이고, 가치를 만들어 내는 공창자입니다.

| 활동 | 내용 | 이어지는 프로덕트 |
| :--- | :--- | :--- |
| **🗺️ 가이드 초대** | 각지의 문화 가이드를 발굴하고, Matsuri 플랫폼에 초대 | [Matsuri App](/docs/ecosystem) |
| **🎪 이벤트 기획** | 독자적인 문화 체험 이벤트나 투어를 기획·게재하고, 집객한다 | [Matsuri App](/docs/ecosystem) |
| **📢 콘텐츠 발신** | J-Times의 기사나 팟캐스트를 제작·확산하고, 새로운 유저를 끌어들인다 | [J-Times](/docs/ecosystem) |
| **🤝 커뮤니티 운영** | 지역이나 온라인 커뮤니티를 키우고, 멤버끼리의 이어짐을 깊게 한다 | GCF 관리 웹 |
| **📊 오퍼레이션** | GCF 관리 대시보드에서 추천 추적·매출 분석·팀 관리를 한다 | GCF 관리 웹 |

하나하나의 활동이 **행동 증명(Proof of Action)** 으로서 온체인에 기록되어, 마이닝 보상으로 직결됩니다.
"좋아하는 일을 하는 것"이, 그대로 경제 활동이 된다——이것이 Matsuri의 공창 모델입니다.

---

:::warning 초대제 ─ 잔여 분량 근소
GCF는 한정된 분량(플래티넘 50명 / 골드 분량 조정 중)이 채워지는 대로, 모집을 종료합니다.
이 권리를 가진다는 것은, Matsuri 경제권의 **공창 파트너**로서, 가장 깊게 관계하는 입장이 되는 것을 의미합니다.

참가를 희망하시는 분은, 기존 멤버로부터의 추천이 필요합니다.
우리는 "수"가 아닌 "뜻"으로 커뮤니티를 키우고자 합니다.
:::
