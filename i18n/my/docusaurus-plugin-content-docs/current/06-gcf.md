# 🤝 GCF——အတူတကွဖန်တီးမှု၏ ဗဟိုကို တာဝန်ယူသော နိုင်ငံတကာ အသိုက်အဝန်း

> **"ပရိသတ်" မှ "အတူတကွ ဖန်တီးသူ" သို့။**
> GCF သို့ ပါဝင်ခြင်းသည် Matsuri စီးပွားရေးဇုန်ကို အတူတကွ ကြီးထွားစေရန် မိတ်ဆွေဖြစ်လာခြင်း ဖြစ်သည်။

Matsuri Coin မြှင့်တင်သော "ယှဉ်ပြိုင်မှုမှ အတူတကွဖန်တီးမှုသို့" ဟူသော ရူပါကို နည်းပညာ သို့မဟုတ် token တစ်ခုတည်းဖြင့် မအောင်မြင်နိုင်ပါ။
ကမ္ဘာတစ်ဝှမ်းတွင် ပြန့်ကျဲနေသော **စိတ်အားထက်သန်သူများ**က မိမိ၏ တည်နေရာတွင် ယဉ်ကျေးမှုနှင့် လူကို ချိတ်ဆက်ကာ ecosystem သို့ အသက်ဝင်စေသည်——ထိုဗဟိုကို တာဝန်ယူသည်မှာ **GCF (Global Community Friends)** ဖြစ်သည်။

GCF သည် ရိုးရိုး "membership" မျှသာ မဟုတ်ပါ။
**လုပ်ငန်း ပူးပေါင်းမှု**အဖြစ် ဝင်ငွေ ခွဲဝေ၍ Matsuri ecosystem ၏ ကြီးထွားမှုကိုယ်တိုင်တွင် ပါဝင်သော **အပြည့်အဝ ဖိတ်ခေါ်မှုအခြေခံ** partner အဖွဲ့အစည်း (DAO) ဖြစ်သည်။

![အတူတကွဖန်တီးမှု ၏ ဝိုင်း ── ရည်မှန်းချက်ကို တူညီသော မိတ်ဆွေများ စုဆုံသော နေရာ](/brand/08_B_traditional_gathering.webp)

## အထူးအခွင့်အရေး ①: တကယ့်လှုပ်ရှားသော Mining (ထုတ်ယူခွင့်)

2027 ဇွန် ၁ ရက်တွင် lockup unlock ဖြစ်မည့် **5 သန်း 5 သောင်း 5 ထောင် (စုစုပေါင်း၏ 61% ခန့်)** MTC သည် ဈေးကွက်သို့ မရောင်းဘဲ **ပါဝင်သူများ ဆု pool** အဖြစ် သိမ်းထားသည်။

:::tip အပြည့်အဝ စွမ်းဆောင်ရည်ဆု
သင်၏ လှုပ်ရှားမှုပမာဏ (ရောင်းအား၊ လူစုအရေအတွက်၊ guide ကြိမ်ရေ) အလိုက် pool မှ အလိုအလျောက် MTC ပေးသည်။
လုပ်ခဲ့တာအတိုင်း တရားမျှတစွာ ဆုရ——၎င်းသည် GCF ၏ ဆု ဒီဇိုင်း။
:::

### Mining Pool Unlock မဖြစ်ခင် (2026〜2027 မေ) ၏ ဆု

Mining pool ၏ MTC ထုတ်လွှတ်မှုသည် 2027 ဇွန်မှ စ၍ဖြစ်သော်လည်း **ထိုကာလမတိုင်မီ ကာလတွင်လည်း ဆု ရှိသည်**:

| ဆု အမျိုးအစား | အကြောင်းအရာ | ပေးငွေ နည်း |
| :--- | :--- | :--- |
| **Referral Commission** | Membership・tour・event စသည်တို့၏ referral ဆု | USDC + MTC (ချက်ချင်း) |
| **Guide ဆု** | ပွဲကျင်းပတိုင်း ကော်မရှင်+ tip | ဘဏ်လွှဲ / crypto |
| **Contribution Score စုဆောင်း** | လှုပ်ရှားမှုအားလုံးကို off-chain တွင် မှတ်တမ်းတင်၍ 2027 ဇွန် pool unlock ဖြစ်သောအခါ **စုဆောင်းထားသော score အလိုက် MTC တစ်ကြိမ်တည်း ခွဲဝေ**ပေးသည် | 2027 ဇွန်တွင် ပြောင်းလဲ |

:::info ဆိုလိုသည်မှာ စောစော စတင်လေ အကျိုးရှိလေ
2027 ဇွန်အထိ score စုဆောင်းရမည့် ကာလ ဖြစ်သည်။ ဤကာလတွင် အများဆုံး လှုပ်ရှားသော member သည် pool unlock ဖြစ်သောအခါ အများဆုံး MTC ကို လက်ခံရရှိမည်။ Referral commission နှင့် guide ဆုကို real-time တွင် ပေးသည့်အတွက် **ဝင်ငွေ သုည ကာလ မရှိပါ**။
:::

ထက်ဝက်ခေတ်၏ အသေးစိတ် schedule ကို [Tokenomics](/docs/tokenomics) တွင် ကြည့်ပါ။

:::tip အစောပိုင်း ပါဝင်သူက ပိုများသော ဆု
Bitcoin ကဲ့သို့ပင် ထက်ဝက်ခေတ်ဖြင့် **အစပိုင်း phase ၏ ထုတ်လွှတ်ပမာဏ အများဆုံးဖြစ်မည်**။ ပထမကာလ (2027〜2029) တွင် တက်ကြွစွာ လှုပ်ရှားသူသည် အများဆုံး MTC ကို ရယူနိုင်သော ယန္တရား ဖြစ်သည်။ ဆုကို "ပါဝင်မှု အစောကြော" မဟုတ်ဘဲ **"ပါဝင်မှု ပမာဏနှင့် အရည်အသွေး"** ဖြင့် ဆုံးဖြတ်သည်။
:::

## အထူးအခွင့်အရေး ②: မြင့်မားသော Referral ဆု

ဈေးနှုန်း မြင့်သော ကုန်ပစ္စည်း (membership၊ VIP tour၊ ပူးပေါင်း အိမ်ခြံမြေ စသည်) ကို မိတ်ဆက်ခြင်းဖြင့် သာမန် affiliate ထက် အဆပေါင်းများစွာ **မြင့်မားသော ဆု (USDC + MTC)** ရယူနိုင်သည်။
Smart contract မှတဆင့် **ချက်ချင်း ပြန်အမ်း**——လူ့လက် မပါဝင်သည့်အတွက် နှောင့်နှေးမှု မရှိ၊ မထင်ရှားမှု မရှိ။

Referral သည် တစ်ကြိမ်တည်း ဆု မဟုတ်ဘဲ မိတ်ဆက်ပေးသူက ecosystem အတွင်း လှုပ်ရှားနေသ၍ **စဉ်ဆက်မပြတ် ဝင်ငွေခွဲဝေ** ဖြစ်ပေါ်သည်။
ဆိုလိုသည်မှာ သင်က ချဲ့ထွင်ခဲ့သော network သည် သင်ကိုယ်တိုင်၏ asset ဖြစ်လာသည်။

## GCF ၏ အခန်းကဏ္ဍ——Member သည် ဘာလုပ်ရမည်နည်း

GCF Member များသည် "ပိုင်ဆိုင်ထား၍ စောင့်ရုံ" ရင်းနှီးသူ မဟုတ်ပါ။
Ecosystem ၏ **လက်တွေ့မြေပြင်**တွင် လှုပ်ရှား၍ တန်ဖိုးကို မွေးဖွားသော အတူတကွ ဖန်တီးသူများ ဖြစ်သည်။

| လှုပ်ရှားမှု | အကြောင်းအရာ | ချိတ်ဆက်သော ထုတ်ကုန် |
| :--- | :--- | :--- |
| **🗺️ Guide ဖိတ်ခေါ်** | ဒေသများ၏ ယဉ်ကျေးမှု guide များကို ရှာဖွေ၊ Matsuri platform သို့ ဖိတ်ခေါ် | [Matsuri App](/docs/ecosystem) |
| **🎪 ပွဲ စီစဉ်** | မိမိ၏ ယဉ်ကျေးမှု အတွေ့အကြုံပွဲ・tour ကို စီစဉ်・တင်ဆက်၊ လူစုဆောင်း | [Matsuri App](/docs/ecosystem) |
| **📢 Content ထုတ်လုပ်** | J-Times ဆောင်းပါး・podcast ထုတ်လုပ်・ပျံ့နှံ့၊ အသုံးပြုသူ အသစ် ခေါ်ဆောင် | [J-Times](/docs/ecosystem) |
| **🤝 အသိုက်အဝန်း လည်ပတ်** | ဒေသ သို့မဟုတ် online အသိုက်အဝန်းကို မွေးမြူ၊ member အချင်းချင်း ချိတ်ဆက်မှုကို နက်ရှိုင်းစေ | GCF Admin Web |
| **📊 လုပ်ငန်းဆောင်ရွက်မှု** | GCF Admin Dashboard တွင် referral ခြေရာခံ・ရောင်းအား ခွဲခြမ်းစိတ်ဖြာ・အဖွဲ့ စီမံခန့်ခွဲ | GCF Admin Web |

လှုပ်ရှားမှုတစ်ခုချင်းစီကို **Proof of Action** အဖြစ် on-chain တွင် မှတ်တမ်းတင်၍ mining ဆုနှင့် တိုက်ရိုက် ချိတ်ဆက်သည်။
"နှစ်သက်သော အရာ လုပ်ခြင်း" က တိုက်ရိုက် စီးပွားရေး လှုပ်ရှားမှု ဖြစ်လာသည်——၎င်းသည် Matsuri ၏ အတူတကွဖန်တီးသော model ဖြစ်သည်။
