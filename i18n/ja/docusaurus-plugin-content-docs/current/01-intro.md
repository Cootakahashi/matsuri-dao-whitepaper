# ⛩️ Matsuri Coin（MTC）へようこそ

> **和の架け橋として、競争から共創へ。**

日本文化を体験する人、届ける人、守る人——
すべての「関わり」が経済的価値になり、文化の未来を支える。
それが **Matsuri Coin（MTC）** です。
