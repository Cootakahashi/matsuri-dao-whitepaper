# ⛩️ Matsuri Coin（MTC）မှ ကြိုဆိုပါသည်

> **和 (ဝ — သဟဇာတ) ၏ တံတားအဖြစ်၊ ယှဉ်ပြိုင်မှုမှ အတူတကွဖန်တီးမှုသို့။**

ဂျပန်ယဉ်ကျေးမှုကို ခံစားသူ၊ ပို့ဆောင်သူ၊ ကာကွယ်သူ—
"ပါဝင်မှု" တိုင်းသည် စီးပွားရေးတန်ဖိုးအဖြစ် ပြောင်းလဲကာ ယဉ်ကျေးမှု၏ အနာဂတ်ကို ထောက်ပံ့ပေးသည်။
ဤသည်မှာ **Matsuri Coin（MTC）** ဖြစ်သည်။

![ယဉ်ကျေးမှုနှင့် နည်းပညာ ပေါင်းစပ်မှု ── Matsuri Coin မှ ပုံဖော်ထားသော torii gate နောက်ကွယ်မှ ကမ္ဘာသစ်](/brand/01_A_brand_hero.webp)
