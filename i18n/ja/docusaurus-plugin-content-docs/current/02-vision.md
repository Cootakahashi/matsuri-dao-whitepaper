## 温故知新——古きを訪ね、新しきを知る

日本文化とは何でしょうか？

縄文時代（約16,000年前）、皇紀2,686年、学術定説でも1,300年以上——起源の解釈はさまざまですが、この地に思いやりと調和の文化が受け継がれてきたことは事実です。

私たちはその叡智に学び、テクノロジーと融合させます。
目指すのは、**関わる人が互いを想い、その想いが巡る社会**です。

---

## 和をもって貴しとなす——調和から価値が生まれる

競争ではなく共創。奪い合うのではなく、関わるすべての人が報われる。

これは理想論ではありません。
ブロックチェーンとスマートコントラクトが、「誰かの善意」に頼らず、**ルールとして公正さを保証する**時代が来ました。
関わるすべての人が透明なルールのもとで報われる経済——それを、私たちは構築します。

---

## 二つの体験の架け橋

私たちが目指すのは、二つの方向からの架け橋です。

**日本人にとって** — 安心して世界と出会える国際体験を。
自分の文化を誇りに思い、それを世界に届けることで報われる仕組みを。

**外国人にとって** — 観光の表面ではなく、日本文化の本質に触れられる体験を。
パッケージツアーでは出会えない「ディープ・ジャパン」への扉を。

異なる文化の人々が共に体験し、共に創り、共に知る。
その交流の積み重ねが、国境を越えた理解を生む——私たちは、そう信じています。

---

## この志を、どう実現するのか

具体的には、3つのプロダクトと1つのトークンで構成される経済圏を構築しています。

| 要素 | 何をするか |
| :--- | :--- |
| **⛩️ [Matsuri](/docs/ecosystem)** | 文化体験の予約・決済・参拝マイニング。訪れるだけでMTCを獲得できるアプリ |
| **📰 [J-Times](/docs/ecosystem)** | 日本文化メディア。読む・聴く・シェアするだけでMTCを獲得 |
| **🤝 [GCF](/docs/gcf)** | ガイドを育て、イベントを企画し、エコシステムを動かす国際コミュニティ |
| **💎 [MTC](/docs/tokenomics)** | これら全てをつなぐデジタル通貨。体験の対価・報酬・投資手段として経済圏を循環する |

志だけでは文化は守れません。持続可能な経済の仕組みが必要です。
