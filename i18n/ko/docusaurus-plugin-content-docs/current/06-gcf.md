# 🤝 GCF — 공창의 중심을 담당하는 국제 커뮤니티

> "관객"에서 "공창자"로.

Matsuri Coin이 내세우는 "경쟁에서 공창으로"라는 비전은, 기술과 토큰만으로는 실현할 수 없습니다.
세계 각지에 흩어진 **열정을 가진 사람들**이, 각각의 장소에서 문화와 사람을 연결하고, 에코시스템에 피를 통하게 하는——그것이 **GCF (Global Community Friends)** 입니다.

GCF는 Matsuri 에코시스템의 **운영에 참여하는 초대제 커뮤니티**입니다.
세계 각지에서 문화 체험을 기획·운영하고, 에코시스템을 현장에서 키우는 동료들이 모여 있습니다.

![공창의 고리 ── 뜻을 같이하는 동료들이 모이는 곳](/brand/08_B_traditional_gathering.webp)

---

:::info 초대제 커뮤니티
GCF는 완전 초대제 커뮤니티입니다. 자세한 내용은 기존 멤버로부터 소개받은 분께 개별적으로 안내드립니다.
:::
