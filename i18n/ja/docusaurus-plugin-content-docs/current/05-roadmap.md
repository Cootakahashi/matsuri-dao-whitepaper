## 🏛️ ガバナンス（DAO）

Matsuri Protocolは、中央集権から徐々に **分散型自律組織（DAO）** へと移行します。
GCFメンバー（プラチナ/ゴールド）は、将来的に以下の重要事項に対する **投票権** を持ちます。

| 投票事項 | 内容 |
| :--- | :--- |
| **💰 資金配分** | 事業収益をどの新規事業やマーケティングに投資するか |
| **⚙️ プロトコル更新** | アプリの手数料率やマイニング報酬率の微調整 |
| **⛩️ 文化認定** | どの祭りや神社を「公式巡礼地」として認定し、資金援助を行うか |
