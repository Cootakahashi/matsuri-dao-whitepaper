# 🤝 GCF——共創の中核を担う国際コミュニティ

> **「観客」から「共創者」へ。**
> GCFに参加するということは、Matsuri経済圏を共に育てる仲間になるということです。

Matsuri Coinが掲げる「競争から共創へ」というビジョンは、技術やトークンだけでは実現できません。
世界中に散らばる**情熱を持った人たち**が、それぞれの場所で文化と人をつなぎ、エコシステムに血を通わせる——その中核を担うのが **GCF（グローバル・コミュニティ・フレンズ）** です。

GCFは単なる「会員権」ではありません。
**事業の共同体**として収益を分け合い、Matsuriエコシステムの成長そのものに参加する**完全招待制**のパートナー組織（DAO）です。

## 会員ティア

| ティア | 役割 | 特権 |
| :---: | :--- | :--- |
| **👑 プラチナ** | 創業パートナー / VIP | **50名限定**。事業の意思決定への参加権、最高ランクの報酬設計 |
| **🥇 ゴールド** | アンバサダー | 実働部隊。活動量に応じた報酬。マイニングと紹介報酬の還元率が最大化 |

プラチナは文字どおり**50名しか存在しない**創業パートナー枠です。
ゴールドメンバーは、現場で動き、つなぎ、広げる——エコシステムのエンジンそのものです。

### プラチナメンバーシップ価格

プラチナ枠は段階的に価格が上昇します。早期に参加するほど、低い価格で創業パートナーになれます。

## 特典①：実働マイニング（採掘権）

2027年6月1日にロックアップが解除される**5億5000万枚（総供給の約61%）**のMTCは、市場に売却されるのではなく**貢献者への報酬プール**として確保されます。

:::tip 完全成果報酬
あなたの活動量（売上、集客数、ガイド回数）に応じて、プールから自動的にMTCが支払われます。
働いた分だけ、正当に報われる——それがGCFの報酬設計です。
:::

### マイニングプール解禁前（2026年〜2027年5月）の報酬

マイニングプールのMTC放出は2027年6月からですが、**それまでの期間も報酬は発生します**：

| 報酬の種類 | 内容 | 支払い方法 |
| :--- | :--- | :--- |
| **紹介コミッション** | 会員権・ツアー・イベントなどの紹介報酬 | USDC + MTC（即時） |
| **ガイド報酬** | イベント開催ごとのコミッション＋チップ | 銀行振込 / 暗号通貨 |
| **貢献スコアの蓄積** | すべての活動がオフチェーンで記録され、2027年6月のプール解禁時に**蓄積したスコアに応じてMTCが一括配布**されます | 2027年6月に変換 |

:::info つまり、早く始めるほど有利
2027年6月まではスコアを貯める期間です。この期間に最も活動したメンバーが、プール解禁時に最大のMTCを受け取ります。紹介コミッションやガイド報酬はリアルタイムで支払われるため、**収入ゼロの期間はありません**。
:::

半減期の詳しいスケジュールは[トークノミクス](/docs/tokenomics)をご覧ください。

:::tip 早期の貢献者がより多く報われる設計
ビットコインと同じく、半減期により**早い段階の放出量が最も多くなります**。第1期（2027〜2029年）に積極的に活動した人が、最も多くのMTCを受け取れる仕組みです。報酬は「参加の早さ」ではなく**「貢献の量と質」**で決まります。
:::

## 特典②：高額紹介報酬

高単価商品（会員権、VIPツアー、提携不動産など）を紹介することで、一般アフィリエイトとは桁違いの**高額報酬（USDC + MTC）**を獲得できます。
スマートコントラクト経由で**即座に還元**——人の手を介さないため、遅延も不透明さもありません。

紹介は一度きりの報酬ではなく、紹介した相手がエコシステム内で活動を続ける限り、**継続的な収益分配**が発生します。
つまり、あなたが広げたネットワークが、あなた自身の資産になるのです。

## GCFの役割——メンバーは何をするのか

GCFメンバーは「保有して待つだけ」の投資家ではありません。
エコシステムの**現場**で動き、価値を生み出す共創者です。

| 活動 | 内容 | つながるプロダクト |
| :--- | :--- | :--- |
| **🗺️ ガイド招待** | 各地の文化ガイドを発掘し、Matsuriプラットフォームに招待 | [Matsuri App](/docs/ecosystem) |
| **🎪 イベント企画** | 独自の文化体験イベントやツアーを企画・掲載し、集客する | [Matsuri App](/docs/ecosystem) |
| **📢 コンテンツ発信** | J-Timesの記事やポッドキャストを制作・拡散し、新しいユーザーを呼び込む | [J-Times](/docs/ecosystem) |
| **🤝 コミュニティ運営** | 地域やオンラインのコミュニティを育て、メンバー同士のつながりを深める | GCF管理Web |
| **📊 オペレーション** | GCF管理ダッシュボードで紹介追跡・売上分析・チーム管理を行う | GCF管理Web |

一つひとつの活動が**行動証明（Proof of Action）**としてオンチェーンに記録され、マイニング報酬に直結します。
「好きなことをする」ことが、そのまま経済活動になる——これがMatsuriの共創モデルです。

:::warning 招待制 ─ 残枠わずか
GCFは限られた枠（プラチナ50名 / ゴールド枠調整中）が埋まり次第、募集を終了します。
この権利を持つことは、Matsuri経済圏の**共創パートナー**として、最も深く関わる立場になることを意味します。

参加をご希望の方は、既存メンバーからの紹介が必要です。
私たちは「数」ではなく「志」でコミュニティを育てたいと考えています。
:::
