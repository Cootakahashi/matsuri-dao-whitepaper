# ⛩️ Matsuri Coin(MTC)에 오신 것을 환영합니다

> **和(화)의 가교로서, 경쟁에서 공창(共創)으로.**

일본 문화를 체험하는 사람, 전하는 사람, 지키는 사람——
모든 "관계"가 경제적 가치가 되어, 문화의 미래를 지탱한다.
그것이 바로 **Matsuri Coin(MTC)** 입니다.
