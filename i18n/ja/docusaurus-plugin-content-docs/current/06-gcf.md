# 🤝 GCF——共創の中核を担う国際コミュニティ

> 「観客」から「共創者」へ。

Matsuri Coinが掲げる「競争から共創へ」というビジョンは、技術やトークンだけでは実現できません。
世界中に散らばる**情熱を持った人たち**が、それぞれの場所で文化と人をつなぎ、エコシステムに血を通わせる——それが **GCF（グローバル・コミュニティ・フレンズ）** です。

GCFは、Matsuriエコシステムの**運営に携わる招待制のコミュニティ**です。
世界各地で文化体験を企画・運営し、エコシステムを現場から育てる仲間が集まっています。
